
# Practice

整理 **Python、資料分析、機器學習與簡易應用** 的練習專案，
涵蓋資料清理、分析建模、影像分類，以及實務導向的小工具與 Web App。

## 專案內容

### 📊 資料分析 / 機器學習

* **practice1_JMA_weather**
  日本氣象廳（JMA）天氣資料分析與整理
* **practice2_hotel_room_rate**
  旅館房價資料分析，探討價格分布與影響因素
* **practice3_regression**
  首爾自行車需求資料，使用迴歸模型進行分析
* **practice4_Image_Classification**
  三類影像分類模型實作，完成訓練與預測流程
* **z_penguins**
  Penguins 資料集分析，包含缺失值處理、類別編碼與視覺化，並使用 Seaborn 和 Matplotlib 繪製資料的關係圖。

### 🛠️ 實用工具 / 應用

* **practice5_photo_rename**
  結合 AI（如 Gemini）進行照片內容辨識與自動重新命名
* **practice6_MySQL**
  使用 MySQL 進行員工餐點資料統計與分析
* **景點餐廳清單 Web App**
  以 Google Apps Script + HTML 從 Google Sheet 讀取資料並動態呈現清單

## 使用技術

* Python
* Pandas / NumPy / Matplotlib
* 機器學習（Regression、Image Classification）
* HTML / JavaScript
* Google Apps Script
* MySQL
